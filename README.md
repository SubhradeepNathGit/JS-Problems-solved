# JavaScript Programming Test Questions

## 1. String Reversal
Write a function that takes a string as input and returns the reverse of the string.

## 2. Remove Elements from Right
Write a JavaScript program to remove specified elements from the right of a given array of elements.

## 3. Reverse a Number
Write a JavaScript function that reverses a number.
- Example: `x = 32243`
- Expected Output: `34223`

## 4. Palindrome Check
Write a JavaScript function that checks whether a passed string is a palindrome or not.
- Note: A palindrome is a word, phrase, or sequence that reads the same backward as forward, e.g., madam or nurses run.

## 5. Alphabetical Order
Write a JavaScript function that returns a string that has letters in alphabetical order.
- Example string: `'webmaster'`
- Expected Output: `'abeemrstw'`

## 6. Count Vowels
Write a JavaScript function that accepts a string as a parameter and counts the number of vowels within the string.
- Note: As the letter 'y' can be regarded as both a vowel and a consonant, do not count 'y' as a vowel here.
- Example string: `'The quick brown fox'`
- Expected Output: `5`

## 7. Type Check
Write a JavaScript function that accepts an argument and returns the type.
- Note: There are six possible values that typeof returns: object, boolean, function, number, string, and undefined.

## 8. Second Lowest and Second Greatest
Write a JavaScript function that takes an array of numbers and finds the second lowest and second greatest numbers, respectively.
- Sample array: `[1, 2, 3, 4, 5]`
- Expected Output: `2, 4`

## 9. Odd or Even Loop
Write a JavaScript for loop that iterates from 0 to 15. For each iteration, it checks if the current number is odd or even, and displays a message on the screen.
- Sample Output: `"0 is even"`, `"1 is odd"`, `"2 is even"`, ...

## 10. Longest Word
Write a JavaScript function that accepts a string as a parameter and finds the longest word within the string.
- Example string: `'Web Development Class'`
- Expected Output: `'Development'`

## 11. Prime Number Check
Write a JavaScript function that accepts a number as a parameter and checks whether it is prime or not.

## 12. Second Lowest and Second Greatest (Duplicate)
Write a JavaScript function that takes an array of numbers and finds the second lowest and second greatest numbers, respectively.
- Sample array: `[1, 2, 3, 4, 5]`
- Expected Output: `2, 4`

## 13. Multiplication and Division
Write a JavaScript program to calculate multiplication and division of two numbers (input from the user).

## 14. Largest of Five Numbers
Write a JavaScript conditional statement to find the largest of five numbers.

## 15. Student Grades Calculator
Write a JavaScript program that computes the average marks of the students in the table, then uses this average to determine the corresponding grade.

| Student Name | Marks |
|--------------|-------|
| David        | 80    |
| Vinoth       | 77    |
| Divya        | 88    |
| Ishitha      | 95    |
| Thomas       | 68    |

**Grade Scale:**
| Range  | Grade |
|--------|-------|
| < 60   | F     |
| < 70   | D     |
| < 80   | C     |
| < 90   | B     |
| < 100  | A     |

## 16. FizzBuzz
Write a JavaScript program that iterates integers from 1 to 100. But for multiples of three print "Fizz" instead of the number, and for multiples of five print "Buzz". For numbers multiples of both three and five print "FizzBuzz".

## 17. Armstrong Numbers
Write a JavaScript program to find the Armstrong numbers of 3 digits.

## 18. Array Check
Write a JavaScript function to check whether an input is an array or not.

## 19. Sort Array
Write a JavaScript program to sort the items of an array.
- Sample array: `var arr1 = [-3, 8, 7, 6, 5, 4, 3, 2, 1];`
- Sample Output: `[-3, -2, 1, 2, 3, 4, 5, 6, 7, 8]`

## 20. Most Frequent Item
Write a JavaScript program to find the most frequent item in an array.
- Sample array: `var arr1 = [3, 'a', 'a', 'a', 2, 3, 'a', 3, 'a', 2, 4, 9, 3];`
- Sample Output: `a (5 times)`

## 21. Swap Case
Write a JavaScript program that accepts a string as input and swaps the case of each character.
- Example: if you input `'The Quick Brown Fox'` the output should be `'tHE qUICK bROWN fOX'`.

## 22. Print Nested Array
Write a JavaScript program that prints the elements of the following array. (Note: Use nested for loops.)
- Sample array: `var a = [[1, 2, 1, 24], [8, 11, 9, 4], [7, 0, 7, 27], [7, 4, 28, 14], [3, 10, 26, 7]];`
- Sample Output: `"row 0"` `"1"` `"2"` `"1"` `"24"` `"row 1"` ...

## 23. Remove Duplicates
Write a JavaScript program to remove duplicate items from an array (ignore case sensitivity).

## 24. Leap Years
Write a JavaScript program to find the leap years in a given range of years.

## 25. Binary Search
Write a JavaScript program to perform a binary search.
- Note: A binary search or half-interval search algorithm finds the position of a specified input value within an array sorted by key value.
- Sample array: `var items = [1, 2, 3, 4, 5, 7, 8, 9];`
- Expected Output: `console.log(binary_Search(items, 1)); //0` and `console.log(binary_Search(items, 5)); //4`

## 26. Sum Array Indexes
There are two arrays with individual values. Write a JavaScript program to compute the sum of each individual index value in the given array.
- Sample array: `array1 = [1, 0, 2, 3, 4];` `array2 = [3, 5, 6, 7, 8, 13];`
- Expected Output: `[4, 5, 8, 10, 12, 13]`

## 27. Find Duplicates
Write a JavaScript program to find duplicate values in a JavaScript array.

## 28. Remove Falsy Values
Write a JavaScript function to remove `null`, `0`, `""`, `false`, `undefined` and `NaN` values from an array.
- Sample array: `[NaN, 0, 15, false, -22, "", undefined, 47, null]`
- Expected result: `[15, -22, 47]`

## 29. Sort Array of Objects
Write a JavaScript function to sort the following array of objects by title value.

## 30. Merge and Remove Duplicates
Write a JavaScript function that merges two arrays and removes all duplicate elements.
- Test data: `var array1 = [1, 2, 3];` `var array2 = [2, 30, 1];`
- Expected Output: `[1, 2, 3, 30]` (Note: order may vary depending on implementation)

## 31. Reverse String Characters

## 32. Redirect to URL

## 33. Async to Promise

## 34. Pluralize

## 35. Async Function Composition

## 36. Minimum Value with Mapping

## 37. Lowercase Object Keys

## 38. String Check
